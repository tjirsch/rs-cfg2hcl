## Reporting a Vulnerability

We take the security of our software seriously. If you suspect that you have found a security vulnerability in this project, please report it immediately.

### How to Report

Please do **not** report security vulnerabilities through public GitHub issues.

Instead, please report them by creating a GitHub Security Advisory (if enabled) or contacting the maintainers directly.

### What to Include

When reporting a vulnerability, please include as much information as possible, including:

*   A description of the vulnerability.
*   Steps to reproduce the vulnerability.
*   Potential impact of the vulnerability.

We will acknowledge your report within 48 hours and will do our best to resolve the issue as quickly as possible.
